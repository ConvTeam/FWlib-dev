You can select one of the following for WOL test:

W5200E01-M3_MACRAW.zip
  - Make and Send Magic Packet for WOL using MACRAW mode of W5200
  - Working on W5200 EVB

ether_wake.zip
  - Make and Send Magic Packet for WOL using WinPcap
  - Required install WinPcap (http://www.winpcap.org/install/)
  - Executable on Windows

W5200_WakeOnLAN.py
  - Make and Send Magic Packet for WOL using Python
  - Required Python Version 2.7
  - Executable on Linux (tested on Ubuntu linux 12.04 with Oracle VM VirtualBox)
